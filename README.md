# Asset Curator

Asset Curator is a powerful asset management app designed to track, organise, and maintain physical assets using QR codes, structured data, and intelligent reporting.

---

## 🚀 Key Features

* QR Code Asset Tracking
* Multi-device Sync (Master Database – Premium and above)
* Advanced Reporting & Insights
* Maintenance Tracking & Forecasting
* Label Printing (Avery & Dymo support)

---

## 📊 Dashboard & Reporting

* Asset status overview
* Location tracking
* Misplaced asset detection
* Advanced reporting insights

---

## 🔧 Maintenance Module

* Track by Hours, Cycles, Distance, or Date
* Routine-based servicing (R1, R2, R3)
* Overdue and upcoming tracking
* Forecasting and analytics

---

## 🔄 Sync System

* Local or Shared Database modes
* Multi-user support
* Safe merge using timestamps
